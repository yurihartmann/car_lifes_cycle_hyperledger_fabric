# Commands Makefile

### `configure-fablo`
- Used to give permission to fablo

## Runs

### `make run-dockers`
- Run API and Frontend docker

### `run-api-docker`
- Run API docker

### `make run-frontend-docker`
- Run Frontend docker

## Stop

### `make stop-dockers`
- Stop API and Frontend docker

### `make stop-api-docker`
- Run API docker

### `make stop-frontend-docker`
- Run Frontend docker
